# GTG Perfumes

A modern, interactive frontend for a perfume brand showcasing products, collections, comparisons, and subscription options.

## Tech Stack

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Responsive design techniques
- Intersection Observer API for scroll animations
